- 개발툴 : Eclipse, Android studio, VSCode
- 사용 언어 : JAVA, MySQL, Android, Html5, CSS3, Javascript, React, Spring Boot
- 사용 기술 : Ajax, Glide, Gson, jQuery, JDBC
